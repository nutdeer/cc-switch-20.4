# CC-Switch 本地文件集

本文件夹是 [CC-Switch](https://github.com/farion1231/cc-switch) **Ubuntu 20.04 适配版**的相关文件整理合集。

CC-Switch 是一个 AI 编程工具管理桌面应用，可统一管理 Claude Code、Claude Desktop、Codex、Gemini CLI、OpenCode、OpenClaw、Hermes Agent 七款 AI 编码助手，提供 50+ 预设 API 提供商、MCP 服务器管理、系统托盘快速切换、云端同步（Dropbox/OneDrive/iCloud/WebDAV）等功能。基于 Tauri 2 + React + TypeScript + Rust 构建。

## 目录结构

| 名称 | 说明 |
|------|------|
| `cc-switch-source/` | CC-Switch v3.16.2 完整源代码 |
| `cc-switch-appimage-refactor/` | AppImage 构建重构相关 |
| `cc-switch-flatpak-build/` | Flatpak 构建文件 |
| `cc-switch-flatpak-repo/` | Flatpak 仓库 |
| `cc-switch-3.9.0-3-deb-extracted/` | 已解压的 v3.9.0-3 旧版 deb 包 |
| `CC-Switch-v3.16.2-extracted/` | 已解压的 v3.16.2 deb 包 |
| `CC-Switch-v3.16.2-Linux-x86_64.deb` | v3.16.2 Debian 安装包 |
| `CC-Switch-v3.9.0-3-Linux.deb` | v3.9.0-3 Debian 安装包 |
